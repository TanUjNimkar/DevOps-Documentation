# DevOps Documentation

A practical collection of DevOps setup guides, commands, and notes based on hands-on experience.
This repository focuses on real-world tools used in CI/CD, containerization, and cloud-native environments.

---

## What This Repository Contains

- Jenkins installation and CI/CD pipeline notes  
- Docker setup and container workflow documentation  
- Jenkins + Docker + Kubernetes integration  
- Kubernetes local setup using Minikube  
- AWS EKS basic setup and concepts  
- Ansible setup and configuration notes  
- SonarQube setup (including Docker-based setup)

Each file is written as a step-by-step reference for learning, revision, and interview preparation.

---

## Who This Is For

- DevOps beginners and learners  
- Engineers preparing for DevOps interviews  
- Developers moving into CI/CD and cloud-native workflows  

---

## How to Use This Repository

- Read the `.txt` files based on the tool you want to learn or set up
- Follow the commands and explanations step by step
- Use it as:
  - a learning reference
  - a quick revision guide
  - interview preparation notes

Recommended learning order:
1. Docker  
2. Jenkins  
3. Kubernetes / Minikube  
4. AWS EKS  
5. Ansible  
6. SonarQube
